## 密涅瓦大學教學方法

密涅瓦大學將大一設定為「基石年（Cornerstone Year）」，所有學生不分科系，都必須接受四大核心能力的嚴格訓練。他們將這四大能力拆解成幾十個可以被反覆練習的思維模組（例如：如何辨識認知偏誤、如何運用演繹邏輯），讓學生在面對未來挑戰時，能像大腦內建的「演算法」一樣自動運用出來：

 1. **批判性思考（Thinking Critically)**
 訓練如何評估各種主張的真偽、分析邏輯推論的破綻、辨識相關性與因果關係的不同，以及如何在資訊不足的情況下做出最佳決策。

 2. **創意思考（Thinking Creatively）**
 跳脫框架解決問題。訓練包含設計思考、啟發式解題法（Heuristics）、如何為複雜的現實狀況建立模型，以及創造新的產品或流程。  
 3. **有效溝通（Communicating Effectively）**
 不只是演識或寫作，還包括如何精準使用語言、掌握聽眾心理，以及運用數據與多媒體來清晰傳達複雜概念。  
 4. **有效互動（Interacting Effectively）**
 將重點放在人際與社會層面，訓練談判技巧、調解衝突、解決道德兩難、團隊協作，以及理解不同利益關係人在系統中的互動模式。
